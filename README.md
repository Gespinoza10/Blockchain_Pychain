# Blockchain | Pychain Ledger 

Streamlit Application-Blockchain created with Python.

## Installation

Before you clone `pychain.py`, make sure you have `streamlit` installed on your `env`.
To install the Streamlit library, check that your dev virtual environment is active, and then run the following command:

```shell
pip install streamlit
```
